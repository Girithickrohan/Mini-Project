# Mini-Project

## JuriAssist - AI - Powered Legal Query Resolution
An Intelligent Legal Assistance System
JuriAssist is a web-based intelligent legal assistant designed to simplify access to the Indian Penal Code (IPC). It helps users search, understand, and manage IPC sections using a clean interface and AI-assisted explanations.

## Problem Statement
Legal information, especially IPC sections, is often difficult for common people to understand due to complex language and lack of accessibility. Searching for the correct section is time-consuming and usually requires legal expertise.

## Proposed Solution
JuriAssist bridges this gap by providing:

    Easy IPC search using keywords or section numbers
    Structured and readable legal content
    AI-based simplified explanations (ClueAI)
    User-friendly and legal-themed UI

## Features

Technologies Used
    Backend: Python, Flask
    Frontend: HTML, CSS, JavaScript
    Version Control: Git, GitHub

## System Architecture (Overview)
    User submits IPC search query
    Flask backend processes the request
    Relevant IPC data is retrieved
    Results are displayed on the dashboard
    ClueAI provides simplified explanations
